1. 子表创建问题

问题描述 - 每次写入数据前都需要判断子表是否存。- 大量设备导致子表创建耗时过长

解决方案 ```java
// 1. 批量预创建子。public void batchCreateSubTables(List devices) {
StringBuilder sql = new StringBuilder();

for (Device device : devices) {
    sql.append(String.format(
        "CREATE TABLE IF NOT EXISTS sensor_data_%s USING sensor_data TAGS ('%s', '%s', '%s', 'temperature');",
        device.getDeviceId(), device.getDeviceId(), device.getDeviceType(), device.getLocation()
    ));
}

tdengineService.execute(sql.toString());

}

// 2. 设备注册时自动创建子。public Device registerDevice(DeviceRegisterDTO dto) {

return device;
